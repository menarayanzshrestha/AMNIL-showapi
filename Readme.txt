This project consist of NODE JS for backend and Reactjs for UI.

#NODEJS
Consist of dependencies :-
    "body-parser"- for getting data from body
    "cors"- for cross origin issue
    "express"- a node web framework
    "node-fetch"- fetch api

#REACTJS
Consist of dependencies :-
    "axios": to fetch api
    "node-sass": to use sass as script 
    "semantic-ui-react": for using UI library
    "sweetalert": for showing message as a model

To run this app:
1.Clone this repo
2.run "npm install" inside main folder as well as inside "client" folder
3.run "npm start" inside main folder as wll as inside "client" folder to run server and client respectively.
